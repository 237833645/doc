# 非工作类技巧

- wget下载整个网站

```bash
wget -c -r -nd -np -k -L -p www.oldlinux.org/Linux.old/
```

- ffmpeg下载m3u8文件

```bash
ffmpeg -i "m3u8列表url" -c copy "存储视频名字.mp4"
```

- vim 技巧

| 描述 | 字段 |
| - | - |
| 全局文件替换字符串 | :%s/archive/mirrors/g |
| 删除光标所在到第一行的所有数据 | d1G或者dgg |
| 删除光标所在到最后一行的所有数据 | dG |
| 删除光标所在到该行最后一个字符 | d$ |
| 删除光标所在出，到该行第一个字符 | d0 |
